This is the final project for Non-clasical optimization class at SGH. The idea of it is to find a solid solution to an optimization problem through implementation of metaheuristic algorithms.
In this case, the problem being solved is finding the shortest path that would make us visit each city exactly one time (travelling salesman problem).
